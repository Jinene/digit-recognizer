# Handwritten Digit Recognizer ✏️

A slightly advanced AI project to recognize handwritten digits (0-9) using a neural network.  
Includes a Streamlit web interface for real-time predictions.

---

## Features
- Train a neural network on MNIST dataset.
- Users can draw digits in a web app and get predictions.
- Simple, interactive deep learning project.
